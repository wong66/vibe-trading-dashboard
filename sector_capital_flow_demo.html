🛠 在你本机复现（完整 ~100 行业净流入排行）

在 a-stock-data v3.4.0 的 industry_comparison() 已扩展 f62/f184 后，调用示例：

from skill import industry_comparison  # 或直接 exec SKILL.md 内函数

data = industry_comparison(20)          # top_n=20
# 按主力净流入降序排（抖音「板块资金流向」同款）
ranked = sorted(data["top"] + data["bottom"],
                key=lambda r: r["main_net"], reverse=True)
for r in ranked[:15]:
    print(f"{r['name']:>8}  主力净流入 {r['main_net']/1e8:.2f}亿  "
          f"净占比 {r['main_net_pct']:.2f}%  涨跌 {r['change_pct']:+.2f}%")

# 抓全量（~100 行业）按净流入排序：
#   params 里 pz=500, fid="f62", po=1  →  直接得净流入排行榜

📌 结论

• 原版 v3.4.0 没有独立的「板块资金流」函数，只有 industry_comparison()（仅涨跌幅+领涨股）和个股资金流。
• 已扩展 industry_comparison() 的 fields 加上 f62(主力净流入) / f184(主力净占比)，现在直接返回板块资金流数据。
• 已实测东财接口返回这两个字段（见上方样本），与抖音「7月16日收盘板块资金流向」同源。
