# 大A投研看板

## 项目定位
搭建一个面向 A 股（大A）市场的投研看板系统，用于投资研究的信息聚合、数据可视化和 AI 量化决策支持。

## 项目结构
```
大A投研看板/
├── Vibe-Trading/          # 核心项目（前端React + 后端Python/FastAPI）
│   ├── agent/             # Python后端
│   │   ├── api_server.py  # FastAPI主入口（端口8898）
│   │   ├── aquant_routes.py      # AI量化决策API路由（4模块）
│   │   ├── signal_engine.py      # 信号引擎（多因子扫描+评分）
│   │   ├── market_state.py       # 市场状态引擎（温度+风格+板块热力）
│   │   └── src/           # Agent/Skills/Swarm/回测/因子等
│   ├── frontend/          # React前端（Vite 端口5899）
│   │   └── src/
│   │       ├── pages/     # AquantSignals/Plans/Review/Delivery等
│   │       ├── components/aquant/  # 量化决策UI组件
│   │       └── lib/       # API/工具库
│   └── A股量化决策/       # 量化决策数据存储（signals/plans/reviews/）
├── astock-peg/            # PEG估值分析（Next.js 端口3000）
└── A股量化决策/           # 根层级量化数据目录
```

## AI量化决策框架（V3 2026-07-01）

### 四模块结构
1. **主线决策**（/aquant/signals）— 多因子信号扫描 + 综合评分 + 排序 + 板块分组
2. **交易计划**（/aquant/plans）— 结构化交易计划管理
3. **复盘雷达**（/aquant/review）— 市场温度 + 命中率 + 板块热度
4. **交割单复盘**（/aquant/delivery）— CSV导入 + 信号匹配 + 胜率统计

### 信号引擎（signal_engine.py）
- **数据源**：a-stock-data 七大层（行情/研报/信号/资金/新闻/基础数据/公告）
- **因子体系**：5大类 18+ 因子
  - 估值：PE百分位、PB百分位
  - 质量：ROE、毛利率、净利率、资产负债率
  - 成长：营收增速、净利增速、EPS 3年CAGR
  - 动量：20日涨跌、60日涨跌、价格位置、量比
  - 资金/情绪：主力资金流、龙虎榜信号、融资变化
- **当日行情**：信号输出附带 `price` / `change_pct` / `change_amount`（来自腾讯实时行情）
- **评分模型**：加权排名百分位归一化 + 板块加成修正
- **选股池**：watchlist（默认10只）/ hs300 / zz500 / custom

### 市场状态引擎（market_state.py）
- **市场温度**：广度维度(涨跌比) + 资金维度(北向) + 情绪维度(指数涨跌)
- **风格检测**：价值/成长/题材/防御 四象限（基于指数相对强弱）
- **板块热力图**：东财行业排名 + 龙头 + 涨跌家数 + 信号密度
- **资金分析**：同花顺北向资金 + 东财主力资金流
- **API 返回**：`/aquant/market/temperature` 的 `temperature` 字段已包含完整指数/广度/北向明细

### 主线决策前端（AquantSignals）
- **市场仪表盘**：六指数卡片（含实时涨跌幅/成交额/PE）+ 市场广度条 + 三维温度 + 北向资金
- **板块热力图**：TOP12 板块网格，每个板块带一句话说明（hover 提示）
- **视图模式**：
  - 评分排序：按综合评分降序
  - 板块分组：按板块分组展示，支持板块筛选下拉框
  - 新信号：今日首次入选或评分跃升的信号
  - 因子维度：按 PE 历史百分位排序（低估值优先）
- **信号行**：排名 + 股票信息 + 当日行情（价格/涨跌幅/涨跌额） + 综合评分 + 因子速览 + 标签
- **信号详情面板**：右侧展开，含 8 维度因子雷达图

### 数据源优先级
| 优先级 | 数据源 | 用途 |
|--------|--------|------|
| 1 | 腾讯财经 | PE/PB/市值/换手率/实时价格/涨跌幅（不封IP） |
| 2 | mootdx(通达信) | K线/财务快照/F10（TCP不封IP） |
| 3 | 同花顺 | 强势股归因/北向资金 |
| 4 | 东财（限流） | 龙虎榜/资金流/行业排名/融资融券 |

### 前端组件（components/aquant）
- **AquantSignals**（主线决策页）：指数行情 + 市场广度 + 北向资金 + 板块热力图 + 信号统计 + 信号列表 + 详情面板
- **MarketDashboard**：六指数卡片 + 市场广度条 + 综合温度 + 三维分维度 + 北向资金
- **SectorHeatmap**：板块热力网格（热度/涨跌幅/龙头/信号数 + 一句话说明）
- **SignalStats**：评分分布 + 资金特征 + 板块分布 TOP3
- **FactorRadarChart**：8 维度因子雷达图（ECharts）
- **SignalDetailPanel**：点击信号展开的右侧详情面板（估值/质量/成长/动量/资金）
